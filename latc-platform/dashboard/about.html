24/7 Interlinking Platform - Dashboard

The 24/7 Interlinking Platform Dashboard is developed and maintained by Michael Hausenblas at the Linked Data Research Centre, DERI, NUI Galway. On the browser-side it uses jQuery (and some plug-ins) and PHP+ARC2 and Apache2 on the server side.

The 24/7 Interlinking Platform Dashboard Web application is powered by the Dashboard software developed in LATC.

LATC 24/7 Interlinking Platform Dashboard is maintained at http://latc.hg.sourceforge.net/hgweb/latc/latc/, release version 0.0 (2010-10-22), The source code is available via the LATC hg platform.
