Dataset Finder

This service uses the LATC Dataset Inventory and Sindice to return URIs of datasets that match a text keyword search.

It takes your input and searches Sindice for domains that publish RDF documents with matching text, then finds matching LOD cloud datasets for those domains.

Results are returned in JSON, for example, a search for Hummingbird will return:
